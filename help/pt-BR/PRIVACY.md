COOKIES E OUTRAS TECNOLOGIAS DE RASTREAMENTO** Cookies da Web Um "Web Cookie" é uma sequência de informações que um site armazena no computador de um usuário e que o navegador do usuário fornece ao site sempre que o usuário envia uma consulta ao site. Usamos Web Cookies para rastrear as páginas que os usuários visitam durante cada sessão do Site, tanto para nos ajudar a melhorar as experiências dos usuários quanto para nos ajudar a entender como o Site está sendo usado. Assim como outras Informações não Pessoais coletadas de usuários do Site, analisamos e divulgamos em forma agregada informações coletadas usando Web Cookies, a fim de ajudar a nós, nossos parceiros e outros a entender melhor como o Site está sendo usado. Não tentamos vincular informações coletadas usando Web Cookies a Informações Pessoais. USUÁRIOS QUE NÃO DESEJAM QUE COOKIES DA WEB SEJAM COLOCADOS EM SEUS COMPUTADORES DEVEM CONFIGURAR SEUS NAVEGADORES PARA RECUSAR COOKIES DA WEB ANTES DE ACESSAR O SITE, COM O ENTENDIMENTO DE QUE CERTOS RECURSOS DO SITE PODEM NÃO FUNCIONAR CORRETAMENTE SEM O AUXÍLIO DE COOKIES DA WEB. USUÁRIOS QUE RECUSAM COOKIES DA WEB ASSUMEM TODA A RESPONSABILIDADE POR QUALQUER PERDA DE FUNCIONALIDADE RESULTANTE. Web Beacons Um “Web Beacon” é um objeto que é incorporado em uma página da web que geralmente é invisível para o usuário e permite que os operadores do site verifiquem se um usuário visualizou uma página da web específica ou um e-mail. Podemos usar Web Beacons no Site e em e-mails para contar usuários que visitaram páginas específicas, visualizaram e-mails e para fornecer serviços de marca compartilhada. Os Web Beacons não são usados para acessar as Informações Pessoais dos usuários; eles são uma técnica que podemos usar para compilar estatísticas agregadas sobre o uso do Site. Os Web Beacons coletam apenas um conjunto limitado de informações, incluindo um número de Web Cookie, hora e data de uma página ou visualização de e-mail e uma descrição da página ou e-mail no qual o Web Beacon reside. Você não pode recusar Web Beacons, no entanto, eles podem ser tornados ineficazes ao recusar todos os Web Cookies ou modificar a configuração do seu navegador para notificá-lo cada vez que um Web Cookie for oferecido e permitir que você aceite ou recuse Web Cookies individualmente. **12.
