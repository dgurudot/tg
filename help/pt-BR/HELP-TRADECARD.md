# **Compreendendo as Operações**

**Observação:** Não é um conselho de investimento. Para a lista completa de isenções de responsabilidade, consulte [https://tradegrub.com/disclaimer](https://tradegrub.com/disclaimer)

- **Compra** - Indica os níveis B1, B2 e B3. A cor verde indica o nível de compra concluído.
- **Venda** - Indica os níveis S1, S2 e S3. A cor verde indica o nível de venda concluído.
- **Parada** - *DEVE* ser definida para todas as operações de forma consistente ou, alternativamente, usar o trailstop (recomendado). Você pode usar alertas se puder agir prontamente. Os alertas também funcionam no mercado estendido e podem ser usados para ordens de venda no mercado estendido, se necessário.
- **Média** - Indica o preço médio de compra com base na média de custo. As operações de swing/longo prazo são compradas usando 15/25/60% em B1, B2 e B3 por padrão.
- **Marca** - Cotação de preço em tempo real com base na fonte de dados selecionada para a conta. O padrão é a cotação de preço *atrasada*
- **Qtd** - Quantidade total para negociação. A compra de quantidade parcial é atualizada em verde e exibida como "Quantidade parcial / Quantidade total"
- **Ganho** - Indica o intervalo entre as vendas realizadas em S1 e S3
- **Tag** - "D" é Day Trade, "S" é Swing Trade, "L" é Long Trade e exibido com um código de cor exclusivo
- **Abertura** - Indica o horário em que a negociação foi aberta
- **Fechamento** - Indica o horário em que a negociação foi fechada. Fechado significa que não há compras adicionais esperadas para este símbolo ou que você já atingiu a meta de venda inicial. Você **DEVE** continuar usando o Stop Price como indicação para sair da operação quando ele ficar negativo.
- O indicador de ponto âmbar ao lado do símbolo indica a posição ativa da conta padrão.

# Ações de Deslizar
- **Ordem** (Deslizar para a Direita) - Permite que você faça ordens com base em uma operação específica e informações da ordem pré-preenchidas (e podem ser ajustadas).
- **Definir Alerta** (Deslizar para a Esquerda) - Permite que você defina alertas típicos para operações e informações pré-preenchidas (e podem ser ajustadas). Você deve definir alertas para cada posição e ordem ativa.
- **Atualizações** (Deslizar para a Esquerda) - Permite que você visualize o histórico de atualizações de operações específicas.
