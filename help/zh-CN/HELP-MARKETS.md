# **市场**

获取股票、加密货币的全面市场信息
- **股票**：顶级金融、科技和服务行业的股票报价
- **加密货币**：概览，比特币、以太坊、Solana 和 Uniswap 价格
- **股票新闻**：美国股市每日热门新闻
- **加密货币新闻**：加密货币每日热门新闻
- **股票筛选器**：具有筛选功能的股票筛选器
- **加密货币筛选器**：具有筛选功能的加密货币筛选器
- **金融市场**：指数、期货、债券、外汇信息
- **加密货币市场**：加密货币市场
- **指数**：包括美国/加拿大、欧洲、亚太地区的全球指数
- **外汇**：主要、次要和特殊外汇汇率
- **今日股票**：今日活跃股票、上涨和下跌股票
- **经济日历**：时间、实际、预测和前值经济日历
