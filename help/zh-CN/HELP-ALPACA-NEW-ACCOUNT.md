# **获取密钥的说明**
- **步骤 1：**从新的单独浏览器窗口转到 [https://alpaca.markets/](https://alpaca.markets/)（或单击上面的链接）并登录到您现有的帐户
- **步骤 2：**从左侧菜单中选择“模拟帐户”，然后选择“模拟概览”，滚动到“您的 API 密钥”，然后单击“查看”
- **步骤 3：**单击“生成新密钥”，它将显示“API 密钥 ID”和“密钥”
- **步骤 4：**刷新或离开此 Alpaca 页面后，您的“密钥”将消失，因此您可能也希望以安全的方式保存它
- **步骤 5：**在 TradeGrub 应用程序中为您的 Alpaca 帐户设置“密钥”和“API 密钥 ID”，以进行模拟交易

# Alpaca 帐户的已知限制
注意：您可以使用替代数据源来避免 #1 - #3 限制（以 TD Ameritrade 为例）。
- 1：不支持延长营业时间。
- 2：符号数量限制为 30。
- 3：净变化和成交量信息不可用。仓位卡不会显示红色或绿色。
- 4：账户活动不可用。订单和仓位在专业订单屏幕上完成后不会立即更新。启用“自动刷新”可自动定期延迟更新仓位和订单。
