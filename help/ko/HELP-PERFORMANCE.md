# **성과**

아래와 같이 계정 성과를 표시합니다.
- **포트폴리오:** 포트폴리오의 총 포지션 수
- **구매력:** 계정의 총 구매력
- **월 매수:** 특정 월의 총 매수 거래 수
- **월 매도:** 특정 월의 총 매도 거래 수
- **월 이자:** 특정 월의 계정 이자
- **월 배당금:** 특정 월 동안 수령한 총 배당금
- **월 이익:** 특정 월의 총 이익
- **월 순 이익:** 총 이익 + 배당금 - 이자
- **YTD 순 매수:** 연초부터 현재까지 순 매수 거래 수
- **YTD 순 매도:** 연초부터 현재까지 순 매도 거래 수
- **YTD 순 이익:** 연초부터 현재까지 순 이익
