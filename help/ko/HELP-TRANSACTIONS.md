# **거래**

기본적으로 지난 60일 동안의 거래를 표시합니다. 작년의 거래만 거래소에서 가져옵니다.
설정을 클릭하여 특정 헤더를 재정렬하고 활성화/비활성화합니다.
헤더를 클릭하여 특정 헤더를 정렬합니다.

**거래(기본값):**
- **날짜:** 거래 날짜
- **유형:** 거래 유형 [판매, 매수, INT(이자), DIV(배당금), WID(인출), DEP(입금) 또는 기타]
- **심볼:** 심볼 이름
- **수량:** 총 수량
- **비용:** 매수 가격
- **총 비용:** 총 매수 가격
- **시간:** 거래 시간
- **수수료:** 브로커 수수료
- **수수료:** 브로커 또는 거래소 수수료 또는 세금
- **ID(선택 사항):** 거래 ID
