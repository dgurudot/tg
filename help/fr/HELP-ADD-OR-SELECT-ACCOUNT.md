# **Ajouter un nouveau compte ou sélectionner un compte**

- Cliquez sur le bouton **Ajouter un compte** pour ajouter un nouveau courtier, une bourse de crypto-monnaies ou un compte de données ou en lecture seule (via Plaid)
- **Sélectionnez n'importe quelle mosaïque de compte** pour modifier ses paramètres. Cliquez sur le bouton Paramètres dans le titre pour mettre à jour les paramètres suivants :
- **Actualisation automatique** : récupère les commandes périodiquement pour les courtiers qui ne prennent pas en charge le flux d'activité en temps réel.
- **Supprimer** : supprime un compte spécifique
- **Reconnexion** : vous permet de vous reconnecter et d'actualiser le compte. Obligatoire pour certains types de comptes tous les 90 jours.
- **Définir la devise** : pour les comptes réservés aux crypto-monnaies, vous pouvez choisir de mettre à jour votre devise par défaut.
- **Définir le compte par défaut** : définit un compte commercial spécifique comme compte par défaut pour les commandes, les positions et le tableau de bord. **Un compte DOIT être défini par défaut.**
- **Définir la source de données** : permet de sélectionner une source de données en temps réel. Vous pouvez sélectionner un compte de courtier spécifique s'il prend en charge les données en temps réel ou une autre source de données.
- **Définir le nom** : permet de définir un nom pour le compte. Longueur maximale : 6
- **Basculer en lecture seule** : active le verrouillage des échanges et arrête de passer des commandes depuis TradeGrub.
