## Согласен с тем, что для автоматической торговли необходимо использовать одно устройство.

Во время выполнения автоматической торговли всегда следует использовать одно устройство.
- Источник данных автоматически переключается на последнее подключенное устройство, что может повлиять на выполнение автоматической торговли при использовании нескольких устройств одновременно для одной учетной записи.
- Для обеспечения постоянного подключения источника данных в режиме реального времени для выполнения автоматической торговли требуется одно устройство.
- Включение автоматической торговли на нескольких устройствах приведет к выполнению одной и той же сделки с каждого устройства. Обратите внимание: используйте только одно устройство для автоматической торговли. Включение автоматической торговли — это настройка, зависящая от конкретного устройства.
